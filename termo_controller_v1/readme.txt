Данный проект будет обладать следующим функционалом:

Необходимо реализовать следующий функционал:
1) Работа PID регулятора (по хорошему реализовать его как в MexBIOS)
2) Поддержка энкодера для изменения температуры "на лету"
3) Способ переконфигурирования устройства "на ходу"

Какие модули долженр содержать данный проект:
1) Модуль глобальных переменных, который будет хранить сделующие переменные
	а) Задание температуры
	б) Температура обратной связи (в нашем случае это DS18B20)
	в) Коэффициенты PID регулятора
	г) Частота ШИМ
	д) скважность и выбор скважности
По сути все глобальные переменные совпадут нулевой версией данного проекта
2) Модуль работы ШИМ выхода (реализован классом)
3) Модуль работы энкодера
4) Модуль аля CLI, который будет реализован через printf()
5) Модуль PID регулятора (будет использована библиотека от Alex Gyver, которую я чуть чуть исправлю)
